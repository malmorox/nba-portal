# nba-stats
Proyecto de estadísticas NBA en Python con [nba_api](https://pypi.org/project/nba_api/), para integrar la librería de Pandas y Matplotlib.

## Instalación

1. Clonar el repositorio
2. Crear entorno virtual: `python -m venv venv`
3. Activar entorno: `source venv/bin/activate`
4. Instalar dependencias: `pip install -r requirements.txt`

## Uso
```python
python src/data_collection.py
```

## Tecnologías
- Python 3.x
- nba_api
- pandas
- matplotlib/seaborn
