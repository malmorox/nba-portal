Proyecto de estadísticas NBA en Python utilizando [nba_api](https://pypi.org/project/nba_api/) junto con Pandas y Matplotlib para análisis y visualización de datos.

## Instalación

1. Clonar el repositorio
2. Crear entorno virtual: `python -m venv venv`
3. Activar entorno: `source venv/bin/activate` (Bash)
4. Instalar dependencias: `pip install -r requirements.txt`

## Uso
```python
python main.py
```

## Tecnologías
- Python 3.x
- nba_api
- pandas
- matplotlib

---

> Este proyecto no está asociado oficialmente con la NBA y los datos se utilizan únicamente con propósitos de aprendizaje y experimentación.
